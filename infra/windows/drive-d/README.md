# Windows Drive D Handoff

Status: safe handoff template.

Windows target:

```text
D:\SIRINX_OS
```

Create this directory on the Windows PC before executing any real sync.

## Recommended Folders

```text
D:\SIRINX_OS\sirinx-co
D:\SIRINX_OS\mirrors
D:\SIRINX_OS\backups
D:\SIRINX_OS\logs
D:\SIRINX_OS\handoff
```

## PowerShell

Use `setup-drive-d.ps1` on the Windows PC. It creates folders only.
