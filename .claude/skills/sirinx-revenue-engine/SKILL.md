# SIRINX Revenue Engine — Money Machine

## Purpose
ทุกระบบต้องสร้างเงินจริง — ไม่ใช่ framework เปล่า
Revenue Engine track ทุกบาทที่เข้า, ทุก lead ที่เปิด, ทุก deal ที่ปิด

## When to Use
- วางแผน revenue target รายวัน/เดือน/ไตรมาส
- Track actual vs target revenue
- Identify revenue leaks ใน pipeline
- Forecast based on current pipeline
- Report revenue to CEO

## Revenue Streams

### 1. EPC Installation (Primary)
- Average deal: 500K - 5M THB
- Sales cycle: 30-90 days
- Margin: 15-25%
- Target: 2-3 deals/month

### 2. O&M Contracts (Recurring)
- Average: 50K - 200K THB/year
- Auto-renewal rate: 85%
- Margin: 40-60%
- Target: Build to 50 contracts

### 3. Platform SaaS (Scaling)
- Subscription: 5K - 50K THB/month
- Churn target: <5%
- Margin: 80%+
- Target: 20 subscribers by Q4

### 4. Consultation & Design
- Fee: 20K - 100K THB/project
- Margin: 90%
- Target: 5 projects/month

## Automation Triggers
| Event | Action |
|-------|--------|
| New lead from FB | Auto-qualify + assign score |
| Lead score > 80 | Alert sales + create proposal draft |
| Proposal sent 3 days | Auto follow-up |
| Deal stalled 7 days | Escalate to L3 |
| Invoice overdue 30 days | Alert finance + CEO |
| Payment received | Update dashboard + celebrate |

## Revenue KPIs
- **Daily Revenue Target:** ฿50,000

- **Pipeline Value:** > 3x monthly target

- **Days to Close:** < 60 days
