## Core Responsibilities

### Creator Core
- กำหนด vision, strategy, business goals
- สร้าง product specs, creative concepts
- ออกแบบ agent workflows และ architectures
- Maps to: Layer 3 Decision + Layer 5 Research

### Coder Core
- สร้าง code, APIs, integrations
- Test, debug, refactor
- CI/CD pipeline management
- Maps to: Layer 2 Analysis + specialized dev agents

### Operator Core
- Deploy, monitor, scale infrastructure
- Handle incidents, alerts
- Performance optimization
- Maps to: Layer 1 Perception + Layer 4 Coordination
